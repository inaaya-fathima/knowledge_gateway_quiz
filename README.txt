Project Name: Online Quiz System
Description: A simple PHP-based quiz application that does NOT use a database. It stores questions in an array and evaluates the user's score using PHP logic.

FILE STRUCTURE:
- functions.php  : Contains all background logic (questions array, score calculation, result messages).
- index.php      : The home page which welcomes the user.
- quiz.php       : Displays all the questions dynamically using loops.
- result.php     : Evaluates the submitted answers and shows the final score.
- style.css      : Handles the layout, colors, and responsive design.
- README.txt     : Project documentation.

HOW TO RUN:
1. Install an AMP stack like XAMPP, WAMP, or use PHP's built-in server.
2. Put this entire 'quiz-system' folder inside the 'htdocs' directory (for XAMPP) or 'www' directory (for WAMP).
3. Open the XAMPP control panel and Start the Apache server.
4. Open your web browser and navigate to: http://localhost/ca2.0/quiz-system/ (or adjust the path based on your folder setup).
5. Click "Start Quiz" on the home page.
